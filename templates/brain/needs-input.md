# Needs Input

Things Claude (or anyone working with you) cannot move on without your input. Track Specific + Impact for each one so you can see what's actually blocking you.

This file is the brake pedal. When `/next` finds an open item here, it stops and asks you for the input. It will not invent an answer to keep moving.

Each entry has:
- A date (when the ask was raised)
- The Specific (what exactly is needed)
- The Impact (what's blocked until you provide it)
- Status: OPEN, PROVIDED, or DROPPED

---
