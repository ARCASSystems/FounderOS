# Flags

> Two types of flags live here. Check this file during the Chief of Staff stall scan at every session start.

---

## Type 1: Role Performance Flags

Things the OS (Claude) should be doing differently. Captured when the founder corrects behavior, expresses frustration, or gives feedback on how the system is working.

### Format
```
[DATE] [FLAG] [what the issue is] | [what should change]
```

---

## Type 2: Friction and Stall Flags

Things in the founder's work that are stuck, stalling, or creating drag. Captured when a priority hasn't moved in 2+ weeks, a commitment was made and not kept, or a decision is being avoided.

### Format
```
[DATE] [STALL] [what's stalled] | [last known status] | [suggested prompt]
```
