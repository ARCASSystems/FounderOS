# Capture inbox

Drop zone for thoughts captured away from the laptop. Any `.txt` or `.md` file that lands here gets swept into the brain the next time you say "catch up" - filed to `brain/rants/` with provenance, names checked against your glossary, nothing rewritten.

How things land here:

- A phone dictation shortcut that writes text files into a cloud-synced folder pointed at this directory
- Exports from a voice-notes app
- Anything you save by hand

You never have to name or format the files. Dump them; the OS files them. Swept files move to `.processed/` so an empty inbox means you are caught up.

Ranked capture channels and how to wire each one: `docs/capture-anywhere.md`.
