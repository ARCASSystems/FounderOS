# traces/

Codex and Claude final-review traces produced during the v1.22 release cycle. Retained for auditability and as a reference for fork maintainers who want to understand what was reviewed and why specific design decisions were made.

- `v121-*` - review traces from the v1.21 cycle
- `v122-*` - Codex final-release review and Claude verification passes for v1.22

These files are not product documentation and are not required for a working install.
